# Personal Finance & Spending Analysis

This project focuses on cleaning, standardizing, and analyzing financial transaction data using SQL and BI tools.

## Tools Used
- SQL
- Power BI
- Excel

## Key Work
- Cleaned and standardized raw transaction data
- Built SQL views for income and expense analysis
- Categorized spending and validated financial data
- Designed dashboards to visualize spending trends and categories

## Status
This project is actively being developed and expanded.
