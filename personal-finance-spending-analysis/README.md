# Personal Finance & Spending Analysis

## Project Overview
This project analyzes real-world personal finance transaction data to identify spending patterns and support data-driven financial decisions.

## Tools
- SQL
- Power BI
- Excel

## Key Work
- Cleaned and standardized raw transaction data
- Built SQL views for income and expense analysis
- Designed BI dashboards to visualize spending trends

## Full Project Repository
The complete project, including code and dashboards, is available here:
